# **Инструкция по установке инструментов для командной работы**

## **1. Установка VS Code**

### Шаг 1: Скачивание
- Перейдите на официальный сайт: https://code.visualstudio.com/
- Нажмите кнопку "Download for Windows" (или для вашей ОС)
- Скачайте установочный файл

### Шаг 2: Установка
1. Запустите скачанный файл
2. Примите лицензионное соглашение
3. На шаге "Select Additional Tasks" отметьте все пункты:
   - ✅ Добавить значок на рабочий стол
   - ✅ Зарегистрировать Code как редактор для файлов
   - ✅ Добавить в PATH
4. Нажмите "Install"
5. После установки запустите VS Code

## **2. Установка расширений в VS Code**

### Способ 1: Через Marketplace
1. Откройте VS Code
2. На панели слева нажмите на значок "Extensions" (или Ctrl+Shift+X)
3. В поисковой строке введите по очереди названия расширений:

**learn-markdown:**
- Введите "learn-markdown" в поиск
- Найдите расширение от "Microsoft"
- Нажмите "Install"

**learn-preview:**
- Введите "learn-preview" в поиск
- Найдите расширение от "Microsoft"
- Нажмите "Install"

**Markdown Preview Mermaid Support:**
- Введите "Markdown Preview Mermaid Support"
- Найдите расширение от "Matt Bierner"
- Нажмите "Install"

### **Способ 2: Через прямые ссылки**
Откройте эти ссылки в браузере - они откроют VS Code и предложат установить расширения:

- [learn-markdown](vscode:extension/ms-learnpack.learn-markdown)
- [learn-preview](vscode:extension/ms-learnpack.learn-preview)
- [Markdown Preview Mermaid Support](vscode:extension/bierner.markdown-mermaid)

## **3. Установка Git**

### Шаг 1: Скачивание
- Перейдите на https://git-scm.com/
- Нажмите "Download for Windows"

### Шаг 2: Установка
1. Запустите установочный файл
2. Следуйте мастеру установки с настройками по умолчанию:
   - **Компоненты**: оставьте все включенными
   - **Редактор по умолчанию**: выберите "Visual Studio Code"
   - **PATH environment**: выберите "Git from the command line and also from 3rd-party software"
   - **HTTPS transport**: выберите "Use the OpenSSL library"
   - **Line endings**: выберите "Checkout Windows-style, commit Unix-style"
   - **Terminal**: выберите "Use MinTTY"
   - **Extra options**: оставьте по умолчанию

### **Шаг 3: Проверка установки**
1. Откройте командную строку (Win+R → cmd)
2. Введите: `git --version`
3. Должна отобразиться версия Git

## **4. Установка TortoiseGit**
### Шаг 1: Скачивание
- Перейдите на https://tortoisegit.org/
- Нажмите "Download"
- Скачайте две вещи:
  1. Основную программу (TortoiseGit)
  2. Языковой пакет (TortoiseGit Language Pack)

### Шаг 2: Установка TortoiseGit
1. Запустите установочный файл TortoiseGit
2. Нажмите "Next" в мастере установки
3. Примите лицензионное соглашение
4. Укажите путь установки (оставьте по умолчанию)
5. На шаге "Select Components" оставьте все включенным
6. Нажмите "Install"
7. После установки нажмите "Finish"

### Шаг 3: Установка языкового пакета
1. Запустите установочный файл языкового пакета
2. Выберите "Russian" или предпочитаемый язык
3. Завершите установку

### Шаг 4: Настройка TortoiseGit
1. На рабочем столе или в любой папке щелкните правой кнопкой мыши
2. Выберите "TortoiseGit" → "Settings"
3. В разделе "General" укажите:
   - Ваше имя и email (должны совпадать с данными Git)
4. Нажмите "OK"

## **5. Проверка работоспособности**

### Проверка Git:
```bash
git --version
git config --list
```
### **Проверка TortoiseGit:**
- Правой кнопкой в любой папке → должны видеть пункты Git

### **Проверка расширений VS Code:**
1. Создайте файл test.md

2. Напишите любой markdown-текст

3. Нажмите Ctrl+Shift+V для предпросмотра

4. Убедитесь, что расширения работают

### **Полезные советы для команды:**
1. Единые настройки: Создайте общий файл с рекомендуемыми настройками VS Code

2. Проверка: Попросите каждого участника сделать тестовый коммит

3. Документация: Сохраните эту инструкцию для новых участников

Если у кого-то возникнут проблемы - проверьте антивирус и брандмауэр, они могут блокировать установку.
